# BI Design 專案目錄

本目錄收錄各 Power BI 報表的設計文件，供初次接觸的訪客快速了解每個專案的設計概念與適用情境。

---

## 專案列表

### [家庭共帳 Power BI 報表](README_finance.md)

| 項目 | 說明 |
|------|------|
| 主檔 | `family_finance.pbix` |
| 設計概念 | 「分類維度 + 明細事實」的雙層 ETL 架構，大/中分類採會計科目式層級編碼（A01、B02…） |
| 資料流 | Power Query 負責讀取與轉型原始 Excel；DAX 在模型層產生日期、年月、個帳等計算維度表 |
| 可能用途 | 家庭記帳追蹤、個人收支分析；架構可能移植至製造業成本報表（料/工/費的科目替換即可） |
| 擴充方式 | 新年度只需在 `fact_accounts\` 放入新 Excel，報表自動合併 |
